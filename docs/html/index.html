AGILab Documentation

Welcome to the AGILab framework documentation. You’ll find everything from quickstarts to API references, as well as example projects.

Shell wrappers for run configs

Mirror any PyCharm run configuration from the command line by regenerating the helper scripts:

python3 tools/generate_runconfig_scripts.py

The command creates executable wrappers in tools/run_configs/, including the right environment variables, working directory, and uv invocation.
