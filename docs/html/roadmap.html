AGILab Roadmap

This roadmap highlights recently shipped capabilities and the next updates planned for the AGILab toolchain. It is intentionally concise so engineers can skim what changed without reading the full changelog.

IDE-independent launch scripts shipped

  • CLI wrappers: Every run configuration now has a generated shell script under tools/run_configs/ grouped by agilab, apps, and components.
  • Regeneration: Keep scripts up to date with python3 tools/generate_runconfig_scripts.py.
  • Docs: README and docs index explain how to use the wrappers, reinforcing that AGILab no longer requires PyCharm.

Environment neutrality shipped

  • README refresh: Messaging now reflects that IDE integrations are optional and highlights CLI-first workflows.
  • Scripting cleanup: Generator normalises environment variables and working directories so scripts run identically to IDE launches.
