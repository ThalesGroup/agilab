AGILab
========

.. image:: Agilab-Overview.svg
   :alt: AGILab Overview
   :align: center

The AGILab Streamlit UI exposes a set of built-in pages plus optional app-specific
dashboards ("page bundles", sometimes called ``apps-pages``).

See:

- :doc:`agilab-help` for the pages overview (core pages + page bundles).
- :doc:`edit-help`, :doc:`execute-help`, :doc:`experiment-help`, :doc:`explore-help` for the built-in pages.
