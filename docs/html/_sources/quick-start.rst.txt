Install Agilab
^^^^^^^^^^^^^^

Quick run (no setup, cross-platform):

.. code-block:: console

   uvx -p 3.13 agilab

Managed workspace (project folder):

.. code-block:: console

   mkdir agi-workspace && cd agi-workspace
   uv init --bare --no-workspace
   uv add agilab
   uv run agilab
