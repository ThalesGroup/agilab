# Data IO 2026 Example

## Purpose

Runs `data_io_2026_project`, a deterministic public demo for autonomous
mission-data decision making.

## Install

```bash
python ~/log/execute/data_io_2026/AGI_install_data_io_2026.py
```

## Run

```bash
python ~/log/execute/data_io_2026/AGI_run_data_io_2026.py
```

## Expected Input

The app seeds a public synthetic scenario under `data_io_2026/scenarios`.

## Expected Output

The run writes decision artifacts under `data_io_2026/results`, including
mission decisions, generated pipeline data, candidate routes, and timeline
tables for the analysis page.
