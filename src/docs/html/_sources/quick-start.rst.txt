Install Agilab
^^^^^^^^^^^^^^^^

Pre-Requisite:
~~~~~~~~~~~~~~

If you want to install on a cluster you need to provide the installer an account that can perform ssh connection
with key authentification.



Linux and MacOS:
----------------

.. code-block:: bash

   unzip agi.zip
   cd agi
   ./install.sh

Windows:
--------

.. code-block:: powershell
   unzip agi.zip
   cd agi
   .\install.ps1

Agilab Execution
^^^^^^^^^^^^^^^^^^

.. code-block:: console

   cd agi
   ./agilab.sh
