AGI-CORE Framework
==================

agi-core is an AI framework for engineering distributed under the BSD License. See [documentation](https://thalesgroup.github.io/agilab).

### Install

```bash

mkdir agi-workspace && cd agi-workspace
uv init && uv add agi-core
```

### Run

```bash
uv run agilab
```
