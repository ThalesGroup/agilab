# DOCX QA Checklist

## Structural checks

- missing or blank figures
- orphaned media files
- duplicated chapter fragments
- broken caption/figure adjacency

## Wording checks

- stale names or acronyms
- inconsistent path terminology
- inconsistent branch/path labels
- export-only artefacts left in body text

## Delivery checks

- DOCX opens cleanly
- figure count looks stable
- captions remain attached
- optional PDF export still works
